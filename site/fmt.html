fmt

An attempt at an auto formatter for simple-expr.

The steps are

  1. Compile the code
  2. Decompile the code again, whitespace will differ
  3. Replace cursor to correct position on selection with sourcemap

Formatter

Reformat the code, moving the cursor will translate to the generated (reformated) code.
